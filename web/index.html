MOUSE (TOUCHPAD)

touchpad → 1351 mouse (port 1)
speed X (horizontal)
speed Y (vertical)
two fingers → right button
invert horizontal (X)
invert vertical (Y)
The DS4/DS5 touchpad drives the GEOS cursor as a 1351 on port 1, so turning it on moves the joystick to Port 2. Lower = slower; the pad is short vertically, so Y is usually a touch faster than X.
