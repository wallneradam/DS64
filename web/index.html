DS64

a game controller as a C64 Ultimate joystick and mouse

C64 CONNECTION

checking...
The Pi reaches the U64 over the network to switch it in and out of joystick mode. Auto-detected when possible; edit to override.

STATUS

no controller WASD OFF
 

CONTROLLER

paired —
Hold SHARE + PS until the lightbar double-flashes, then press.

JOYSTICK PORT

which control port the C64 sees
Port 1 turns the touchpad mouse off — the mouse uses port 1.

MODE

WASD switching
auto-off after idle s
 

EXTRA BUTTONS

PS button → U64 menu (F10)
circle → left (back in menu)
options → F1 key
share → swap Joy1/Joy2

MOUSE

1351 mouse on port 1 (touchpad and/or USB)
Master switch for both the touchpad and a USB mouse. Turning it on moves the joystick to Port 2, since the 1351 takes port 1.

MOUSE (USB)

A USB mouse is plugged into the Pi — it drives the 1351 on port 1, so the touchpad steps aside.
speed
use the touchpad at the same time

MOUSE (TOUCHPAD)

speed X (horizontal)
speed Y (vertical)
two fingers → right button
invert horizontal (X)
invert vertical (Y)
The DS4/DS5 touchpad drives the GEOS cursor as a 1351 on port 1. Lower = slower; the pad is short vertically, so Y is usually a touch faster than X.
